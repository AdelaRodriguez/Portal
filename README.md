# Mi Tienda — MVP (GitHub Pages)

Sitio estático para lanzamiento rápido (landing + catálogo). Deploy en GitHub Pages.

Estructura:
- index.html
- styles.css
- assets/images/

Cómo desplegar (resumen):
1. Subir archivos al repo.
2. Settings > Pages -> Source: main / root.
3. (Opcional) Añadir CNAME con dominio propio y configurar DNS.
4. Activar "Enforce HTTPS".

Notas:
- Reemplaza COUNTRYCODENUMBER en los enlaces WhatsApp por tu número internacional sin + (ej. 34123456789).
- Reemplaza los enlaces de pago por Payment Links de Stripe o PayPal.Me.
- Optimiza imágenes y prueba una compra real de prueba.
